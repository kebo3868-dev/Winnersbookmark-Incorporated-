# Winnersbookmark Virtual Secretary AI — Product Spec (MVP)

## Positioning
Build **Winnersbookmark** as a modular executive-operations AI:
- Virtual Secretary
- Executive Assistant
- Automation Brain

Not a reactive chatbot. Must be proactive.

## 3-Layer Operating Model

### 1) Information Layer
- Calendar
- Email
- Tasks
- Notes
- Contacts

### 2) Decision Layer
- Prioritization
- Scheduling optimization
- Delegation detection
- Meeting value analysis

### 3) Execution Layer
- Send emails
- Book/reschedule meetings
- Organize task queues
- Create reports
- Trigger automations

## Core Purpose
Protect user time, focus, and productivity by reducing admin workload and increasing execution quality.

## Primary Capabilities
1. Calendar management (conflicts, focus blocks, rebalancing)
2. Email triage and drafting
3. Task prioritization (high/med/low/delegatable)
4. Daily briefing generation
5. Meeting preparation briefs
6. Notes + knowledge vault categorization
7. Voice-command compatible command layer

## UI Modules
- Daily Briefing
- Calendar View
- Task Manager
- Email Summary
- Notes & Ideas

Style: clean, minimal, professional, productivity-first.

## Technical Direction
- Frontend: React + Next.js + Tailwind
- Backend: Node.js services + optional Python AI workers
- Integrations: Google Calendar, Gmail, Slack, Notion, Zapier

## Suggested Agent Split
- Scheduler AI
- Communication AI
- Strategy AI

- System architecture
- Backend design

- Prompt strategy
